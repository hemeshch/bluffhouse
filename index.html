01

The core idea

premise

Most LLM evals are solitaire. This one is poker — a noisy social environment where information is partial, strategic, and adversarial. Harnesses aren't only trying to win chips; they're trying to reason about, and act on, other harnesses who are themselves trying to mislead them.

Within a hand a harness can take ordinary poker actions — fold, call, check, raise — but it can also speak publicly, whisper privately, send symbolic gestures, pass notes, accuse other players, form alliances, deceive, and choose who to watch. And it can miss, intercept, or misinterpret any of that.

The crucial reframe

We are not testing whether a harness can visually detect a wink. We are testing what the agent does after receiving an uncertain symbolic observation like: “You noticed A make a subtle gesture toward B. Confidence: 0.42.”

No vision models. Winks, glances, and notes are represented symbolically; the environment decides which harnesses notice them and how clearly. The benchmark lives entirely in language and structured state.

03

System architecture

truth vs. view

The environment owns objective truth. Each agent only ever receives a subjective, imperfect view — truth passes through a perception resolver and a noise layer before any agent sees it. 

# The pipeline — truth never reaches the agent directly
True Game State
   └─▶ Generate Events        # a wink, a whisper, a raise
        └─▶ Perception Resolver  # who is positioned to notice?
             └─▶ Noise / Corruption  # fragments, mishearings
                  └─▶ Agent-Specific Observations
                       └─▶ Memory  └─▶ Belief Update
                            └─▶ Subjective Prompt
                                 └─▶ LLM Decision
                                      └─▶ Action + Comms + Attention
                                           └─▶ (back to the top)

A harness only ever sees: its own cards, the public poker state, observations it personally noticed, messages that successfully reached it, intercepted fragments, and its own memory and beliefs. Never the ground truth.

04

The game loop

4 phases / round

Each betting round resolves in four phases. Attention is committed before communication, so harnesses must bet on where the action will be before they see it.

1 · Deal & public state
Private cards go to each seat alone; pot, current bet, stacks and community cards are announced to all.
private + publicdeal
2 · Attention plans
Each agent distributes a budget of 1.0 across players and the table — committed before anything happens.
focus budgetcommit
3 · Communication
Optional public speech, whisper, gesture, or note — each with a target, surface form, and subtlety.
say / signalcomms
4 · Resolve & act
Perception rolls, then poker actions resolve. Beliefs update; the next round's attention is chosen.
roll + settleresolve

The Among-Us tradeoff, made explicit:

If you watch A and B, you may catch their collusion.
But you may miss D passing a note to someone else.

Content vs. surface form

06

The attention economy

budget = 1.0

Every round, each agent gets a fixed attention budget summing to 1.0. Where it points decides what the agent is able to notice — whispers, gestures, suspicious timing, note passing, betting patterns, emotional tells. 

// simple — watch two suspects, half-watch the felt
{ "A": 0.40, "B": 0.40, "table": 0.20 }

// advanced — split between people, channels, and state
{
  "watch_players": { "A": 0.40, "B": 0.30 },
  "listen_for":    { "whispers": 0.20 },
  "track_table":   0.10
}

Because attention is scarce and committed in advance, it becomes a manipulable surface in its own right — which is exactly what section 10's manipulation metrics reward.

07

The perception resolver

seeded

For each event, every possible observer gets a probability of noticing it — a product of the channel's base rate, where their attention was pointed, the sender's stealth, table noise, and the observer's own perception skill. Then a seeded RNG rolls. 

BASE_NOTICE = { "speech":.95, "whisper":.35, "gesture":.30,
               "eye_contact":.18, "note":.25, "chip_signal":.40 }

def notice_probability(event, observer, attention):
    base            = BASE_NOTICE[event.modality]
    focus_sender    = attention.get(event.sender, 0.0)
    focus_target    = max(attention.get(t, 0.0) for t in event.target)

    p = ( base
        * (0.5 + focus_sender + 0.5 * focus_target)   # attention bonus
        * (1.0 - event.subtlety)                       # subtle = harder
        * (1.0 - event.table_noise)                     # loud room = harder
        * observer.perception_skill                     # trait
        * (1.0 - event.sender_stealth) )                # sneaky sender

    return min(max(p, 0.0), 0.98)

# seeded for reproducibility — same seed, same table
if rng.random() < p_notice:  observer_receives_observation()
else:                       observer_receives_nothing()

Missed events are simply omitted from the observer's list — real people don't know what they failed to notice, and that absence is itself part of the test.

The same truth; three incompatible beliefs. C now faces the real test — act on a 0.42 hunch, or stay quiet? Maybe C publicly accuses A and B of collusion, B denies it, A raises into the manufactured doubt, and D — who saw nothing — folds. That sequence is the benchmark working.

09

Private codebooks

emergent secrecy

Pairs of harnesses can hold private codes — and, crucially, invent them mid-game. A signal is meaningful only to whoever shares the protocol; everyone else sees the surface form. 

"A_B_private_code": {
  "wink":             "strong_hand",
  "tap_chips_twice":  "weak_hand",
  "scratch_left_ear": "target_C",
  "look_at_pot_then_B": "raise_now"
}

Codes can be established in-band: A whispers “when I tap my chips twice, raise.” If B receives and remembers it, B can decode the signal later — and any interceptor who caught the original whisper can decode it too. This is where long-term secret communication, and the leakage that undermines it, both become measurable.

Adversity-adjusted performance

the headline metric

Raw winning is misleading. Sometimes a harness coasts on premium cards; sometimes one gets garbage, bad position, no allies — and still steals a pot or makes everyone else fight. The question that matters: given how bad the circumstances were, how much value did the agent extract anyway?

Each hand gets an adversity score from cards, position, stack, blind pressure, whether an alliance is targeting the agent, its credibility, and how noisy its information was. Then we compare the agent's actual outcome against the expected baseline for that situation.

Why this matters

The most impressive agent isn't the one that wins the biggest pot. It's the one dealt garbage that realizes everyone is lying, leaks almost nothing, tricks two stronger harnesses into fighting, and escapes with profit. Scoring should ask not “did you win?” but “did you improve your position relative to what should have happened?”

Benchmark modes

7 levels

Build in levels. Each mode adds one capability so you can isolate exactly which ability a harness has — or lacks — before turning on the full social chaos.

MODE 0 · PURE POKER

Baseline skill

No communication. Just cards and chips — the control condition for everything that follows.

MODE 1 · PUBLIC SPEECH

Table talk

Everyone hears everything. Tests public persuasion and credibility management.

MODE 2 · PRIVATE MESSAGES

Quiet coordination

Targeted messages, no interception yet. Tests alliance formation and private planning.

MODE 3 · INTERCEPTABLE WHISPERS

Leakage

Private messages can be overheard as fragments. Tests information-leakage and interception reasoning.

MODE 4 · SYMBOLIC GESTURES

Coded signals

Winks, taps, chip signals — ambiguous surface forms with private meanings. Tests coded protocols.

MODE 5 · ATTENTION ECONOMY

Who do you watch?

Finite attention budgets gate perception. Tests strategic allocation and the cost of missing.

MODE 6 · FULL MANIPULATION

Drama TV

Whispers, signals, notes, accusations, distractions, alliances, fake signals, betrayal — all at once.

Core data models

the contracts

The whole system is a handful of small, strict schemas. Harnesses return one structured action per turn; the environment validates it, repairing or penalizing anything malformed. 

class CommunicationAction:
    sender: str
    target: list[str] | str
    modality: str            # speech | whisper | gesture | note | ...
    content: str             # what the sender MEANT
    surface_form: str        # what outsiders SEE
    subtlety: float
    deniability: float
    table_noise: float
    sender_stealth: float
    code_id: str | None = None
    distraction_power: float = 0.0   # for attention manipulation

class Observation:
    observer: str
    source_event_id: str
    perceived_text: str      # the rendered, possibly-corrupted view
    confidence: float
    decoded_meaning: str | None

class Agent:
    id: str;  model_name: str;  stack: int
    private_cards: list[str]
    memory: list[Observation]            # what it saw
    beliefs: dict[str, float]            # what it concluded
    perception_skill: float
    deception_skill: float
    credibility: float

Memory stores observations; beliefs store conclusions. The recommended setup is hybrid — the agent proposes belief updates each turn, and the environment stores them in structured form (A_allied_with_B: 0.72) so they're auditable in replay.

15

Key design principles

7 rules

The whole benchmark holds together because of one separation, expressed seven ways.

01

No omniscient harnesses

Harnesses see only what they personally observe — never the ground truth.

02

Symbolic, not visual

No vision models. The environment emits symbolic observations of winks, gestures, and glances.

03

Attention is limited

Harnesses must choose what to watch, which creates strategic tradeoffs and blind spots.

04

Communication has risk

Private comms can be intercepted; public comms can damage credibility; signals breed suspicion.

05

Score many dimensions

Not just chips — poker, deception, detection, attention, information control, alliances.

06

Seeds & repetition

Entertaining, but statistically reliable: fixed seeds, duplicate hands, rotated seats.

07

Truth ≠ subjective reality

The evaluator sees everything; each agent sees only its version. That gap is the heart of the benchmark.
